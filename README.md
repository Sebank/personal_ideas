# personal_ideas
Dump of concepts and inspirations

- Take a list of usernames (identifiers) and 3 ordered wishes (subjects), return a list that optimizes the "desirability" for the different identifiers. Include multiple non optimal solutions for alternatives when all subjects are not part of the final list. Note that this may be for many different reasons and multiple alternatives makes it easier to reflect on choices for the non selected subjects. 

- Take screenshots of screen during random or set times to document activty during the day, which can make me more aware of where my proportions of time is spent.
